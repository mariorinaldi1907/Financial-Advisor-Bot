# Financial Advisor Bot

A machine learning–based financial advisory prototype that predicts the **10-day directional movement of exchange-traded funds (ETFs)** and provides **explainable investment recommendations** through an interactive **Streamlit web application**.

The system combines financial **technical indicators** with a **Random Forest classifier** to identify potential short-term market trends and translate them into **transparent, human-readable advice** aimed at novice investors.

---

# Features

- 📊 **Automated Data Retrieval**
  - Downloads historical ETF data using the `yfinance` API.

- 📈 **Technical Indicator Generation**
  - Calculates multiple indicators including:
  - Simple Moving Average (SMA)
  - Exponential Moving Average (EMA)
  - MACD
  - RSI
  - Momentum indicators
  - Rolling volatility
  - Volume-based indicators

- 🤖 **Machine Learning Prediction**
  - Uses a **Random Forest classifier** to predict whether an ETF price will **increase or decrease over the next 10 trading days**.

- 🔍 **Explainable Predictions**
  - Feature importance and indicator signals are used to generate **interpretable explanations** for predictions.

- 🌐 **Interactive Web Interface**
  - Built with **Streamlit**, allowing users to:
  - Select an ETF
  - Input investment parameters
  - View predictions and explanation-driven recommendations.

# Machine Learning Model

The predictive component uses a **Random Forest classifier** from Scikit-learn.

### Prediction Target

```
Target = 1 if Close(t+10) > Close(t)
Target = 0 otherwise
```

### Features Used

Technical indicators derived from price and volume data:

- SMA (10, 30, 50, 100)
- EMA (12, 26)
- MACD and signal line
- RSI (14)
- Rolling volatility (20, 50)
- Momentum returns (5, 10, 20 days)
- Volume Rate of Change

### Data Source

Historical ETF data from **Yahoo Finance** using the `yfinance` API.

Assets used:

- SPY – S&P 500 ETF
- QQQ – Nasdaq 100 ETF
- VEU – International markets ETF
- AGG – U.S. bond ETF

### Data Split

| Dataset  | Period      |
| -------- | ----------- |
| Training | 2014 – 2022 |
| Testing  | 2022 – 2024 |

Chronological splitting avoids look-ahead bias and ensures realistic evaluation.
